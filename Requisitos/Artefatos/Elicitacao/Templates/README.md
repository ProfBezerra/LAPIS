# 📁 Templates de Artefatos de Elicitação

Esta pasta reúne os templates oficiais para criação dos principais artefatos de requisitos e elicitação do projeto.

## Como usar

- Copie o template desejado para a pasta do seu artefato.
- Preencha cada seção conforme as orientações do próprio template.
- Consulte exemplos e o guia de elaboração para dúvidas.
