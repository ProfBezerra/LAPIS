# 📖 Template de Glossário de Termos

> Use este template para registrar e padronizar os termos relevantes do projeto. Adicione, edite ou remova linhas conforme necessário.

## Instruções
- Liste todos os termos, siglas ou expressões relevantes para o projeto.
- Use definições claras, objetivas e sem jargões desnecessários.
- Sempre que possível, cite a fonte ou contexto do termo.
- Atualize o glossário continuamente durante o projeto.

| SRE              | Site Reliability Engineering, prática de engenharia de confiabilidade.    | Google SRE Book                 |
