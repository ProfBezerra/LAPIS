# Gestao de Configuracao

Esta pasta reune materiais de apoio para configurar e operar a gestao do trabalho no GitHub (issues, projetos, sprints e quadros).

## Objetivo

Padronizar como a configuracao do projeto e feita, facilitando:

- onboarding de novos participantes;
- organizacao do fluxo de trabalho;
- rastreabilidade de atividades e entregas.

## Estrutura da Pasta

- `TUTORIAL_GITHUB_PROJECTS.md`: guia principal para configurar e usar GitHub Projects no contexto do repositorio.
- `image/`: imagens usadas na documentacao e nos tutoriais desta pasta.

## Como Usar

1. Leia o tutorial principal para entender o passo a passo de configuracao.
2. Siga as orientacoes para criar ou ajustar boards, colunas, sprints e automacoes.
3. Consulte a pasta de imagens quando precisar de apoio visual durante a configuracao.

## Relacao com o Repositorio

A documentacao desta pasta complementa o material principal do projeto e apoia a aplicacao pratica do modelo LAPIS no acompanhamento do trabalho.
